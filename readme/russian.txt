[center][b]Microdata 4 SMF[/b][/center]

Мод добавляет различные виды семантической микроразметки к страницам SMF.

Разметка хлебных крошек для Google.
Разметка OpenGraph для первого сообщения на странице (поддерживается FaceBook, ВКонтакте и прочие).
Разметка TwitterCards для первого сообщения на странице (поддерживается Twitter).

Подробное описание [url]https://github.com/realdigger/SMF-Microdata-OpenGraph[/url]
